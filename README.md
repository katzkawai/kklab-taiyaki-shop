# 浅草 金たい焼き — Asakusa Kin Taiyaki

浅草のたい焼き屋「金たい焼き」の紹介サイト。日英バイリンガル対応、シングルページで構成された静的サイトです。

## 構成

```
├── index.html       # メインページ（日英バイリンガル）
├── css/
│   └── style.css    # スタイルシート（和モダンデザイン）
├── js/
│   └── main.js      # 言語切替・アニメーション・フォーム制御
└── images/          # 画像用ディレクトリ
```

## 機能

- **日英言語切替** — 右上トグルボタン（localStorage保存）
- **5セクション構成** — Hero / こだわり / メニュー / アクセス / お問い合わせ
- **レスポンシブ対応** — スマートフォン・タブレット対応
- **スクロールアニメーション** — IntersectionObserver によるフェードイン
- **Google Maps埋め込み** — アクセス情報
- **お問い合わせフォーム** — 画面表示用

## 技術スタック

- HTML / CSS / JavaScript（フレームワーク不使用）
- Google Fonts（Noto Serif JP / Noto Sans JP）
- GitHub Pages で公開
